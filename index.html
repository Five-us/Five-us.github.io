我的第一篇博客

使用hexo搭建个人博客

第一步 安装node.js

官网：https://nodejs.org/zh-cn/

选择长期支持版本，傻瓜式安装，最后在系统环境变量中看一下是否有node的路径

第二步 安装hexo

因为直接使用npm安装，速度太慢，这里先安装淘宝源镜像：

npm install -g cnpm –registry=https://registry.npm.taobao.org

查看版本检查是否安装无误

使用cnpm安装hexo，仍然是查看版本检查安装无误

cnpm install -g hexo-cli

第三步 使用hexo搭建博客

新建一个文件夹，我建的文件叫blog，在文件夹中使用Git Bash Here，输入hexo init

报错：

解决方法：

https://www.jianshu.com/p/3ca2d0f049e7

重新hexo init，成功
